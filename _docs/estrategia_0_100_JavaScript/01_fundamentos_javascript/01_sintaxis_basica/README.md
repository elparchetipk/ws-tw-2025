# 📝 Tema 1: Sintaxis Básica de JavaScript

## 🎯 Objetivos

- Comprender la estructura básica de JavaScript
- Dominar la declaración de variables
- Aprender a usar comentarios efectivamente
- Familiarizarse con la consola del navegador

## 📚 Conceptos Clave

### Variables

```javascript
// Declaración de variables
var nombre = 'Juan';        // Forma antigua (evitar)
let edad = 25;              // Forma moderna (preferida)
const PI = 3.14159;         // Constante (no se puede cambiar)
```

### Comentarios

```javascript
// Comentario de una línea

/*
  Comentario de
  múltiples líneas
*/
```

### Console.log

```javascript
console.log('Hola, mundo!');
console.log(edad);
console.log('Mi nombre es', nombre);
```
